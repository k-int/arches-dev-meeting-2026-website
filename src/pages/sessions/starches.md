The presentation is showing work done by Flax and Teal to create a product that allows arches users to deploy a static site that provides both text and geospatial search without the need to directly connect to arches.

This introduces some new technology to allow for efficient searching via a static site and showcases a new way to display arches data.

The presentation is for both developers and end users but mainly focused towards development.

This impacts the community by introducing new tools and ways to work with arches data.

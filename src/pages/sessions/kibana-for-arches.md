This talk covers using Kibana with Arches, including setup, limitations with nested data structures, and dashboards. Kibana is an open-source interface for data in Elasticsearch and can be used by the Arches community to build dynamic, interactive dashboards and gain additional insights from Arches data. This presentation would provide practical insights for anyone who wants to set up Kibana for their Arches instance, feed into the Arches documentation on Kibana dashboards, and invite feedback from other developers on similar experiences of working with nested data structures.
